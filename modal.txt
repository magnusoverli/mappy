## EntryEditModal - Visual Description

### Exact Visual Details

**Modal Window:**
- 70% transparent black backdrop with 4px blur effect
- White/light colored rectangular dialog box with rounded corners (8px radius)
- Smooth slide-up animation when opening (50px upward movement over 0.3 seconds)
- Takes exactly 65% of viewport width and 85% of viewport height

**Header Bar:**
- Dark toolbar spanning full width at top
- Bold "Mappy" text in brand font on far left
- Search input field (max 300px wide) next to brand name
- Layer information text in center showing "[Layer Name] - Editing [Targets/Sources]"
- X close icon button on far right

**Main Content Area (Split Layout):**

**Left Panel (Data Table):**
- Takes up remaining space after right panel's 450px width
- 16px padding on all sides
- **Empty State**: Centered vertically and horizontally with two lines of gray text:
  - Large text: "No entries to edit"
  - Smaller text: "Add some entries using the controls on the right to get started."
- **With Data**: Scrollable table with 36px tall rows, checkboxes for selection, editable cells

**Right Panel (Fixed 450px wide):**
- Thin gray vertical border on left edge
- 16px padding, 16px gaps between sections

**Section 1 - "Add Entries":**
- Bold section title
- Three number input fields stacked vertically:
  - "Quantity" (0-1000 range)
  - "First Key" (0-9999 range) 
  - "Offset" (full integer range)
- Blue "ADD BATCH" button with plus icon
- Red outlined "DELETE SELECTED" button with trash icon (grayed out when nothing selected)

**Section 2 - Transform Controls:**
- Thin gray horizontal divider line
- Title showing "Transform Selected Entries (X selected)"
- **When <2 selected**: Gray text "Select 2 or more entries to enable transformations."
- **When 2+ selected**: Dropdown menu labeled "What do you want to do?" with options:
  - "Shift Keys Up/Down"
  - "Shift Values Up/Down" 
  - "Number Values in Order"
  - "Set Same Value for All"
- Small gray help text below dropdown explaining the selected option

**Dynamic Input Fields (based on dropdown selection):**
- **Shift Keys/Values**: "Move by" number field, red "Conflicts detected" text if applicable
- **Sequential**: Two fields - "Start counting from" and "Count by", each with gray hex conversion text below
- **Fixed Value**: Text field for hex value with red error styling if invalid, plus two small buttons "All 0" and "All F"

**Preview Box:**
- "What will happen:" subtitle
- Light gray bordered rectangle containing:
  - Monospace font entries showing "oldKey = oldValue â†’ newKey = newValue" 
  - Right-pointing arrow icons between old and new values
  - "...and X more entries" text if more than 10 changes
  - "No changes" text if no modifications
- Blue "Apply to X entries" button below (grayed out if conflicts or no changes)

**Footer:**
- Thin gray horizontal border at top
- 16px padding
- Two buttons right-aligned with 16px gap:
  - Gray "Cancel" button
  - Blue "Save" button (grayed out if no changes made)

**Text Styling:**
- All number inputs use monospace font
- Preview entries use monospace font at 0.8rem size
- Help text uses smaller gray font
- Error text uses red color
- Section titles use bold weight
